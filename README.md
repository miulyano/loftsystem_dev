# loftsystem_dev
>loftsystem_dev - graduation project intensive by node.js

### Copy & Start project

```shell
$ git clone https://github.com/morecodemore/loftsystem_dev.git
$ cd loftsystem_dev.git
$ npm i
```

#### for MacOS:
```shell
$ brew install mongodb
$ npm run dev
```

#### for Win:
```shell
* run local mongodb *
$ nodemon ./server/app.js * or * $ node ./server/app.js
```
